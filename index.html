Virtual-Environment

Install Pipenv Package

In this project, we are using Pipenv for dependency management. Thus, you need the pipenv CLI tool installed on your computer: - open cmd or powershell - pip install pipenv

First install of Environment

As soon this package is installed, you can install all required packages for this project by executing the following command inside the root project folder: - open cmd or powershell - cd /your/local/github/repofolder/ - pipenv install - restart VSCode - choose the newly created gfs-functions virtual environment python interpreter (for help see this tutorial

Environment already installed (update dependencies)

If your environment exists and you only want to update the dependencies use these steps: - open cmd or powershell - cd /your/local/github/repofolder/ - pipenv sync

Add new Dependency

If you need another dependency, not yet defined in the Pipfile, you can install it using this command and it will also be added to the dependency list. - open cmd or powershell - cd /your/local/github/repofolder/ - pipenv install <package>
